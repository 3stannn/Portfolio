## ğŸ¨ Customization

### Changing Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --bg-primary: #1a1a1a;
    --text-primary: #D9D9D9;
    --text-accent: #007BFF;
```

1. **Projects**: Edit the project cards in `projects.html`
2. **Skills**: Update skill tags in `home.html`
